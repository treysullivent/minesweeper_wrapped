Please use the included jar file to run the project

OR 

To build the Minesweeper project from the code.

1. Unzip the directory and move the minesweeper_wrapped folder to your workspace location
2. Import the minesweeper_wrapped folder as a gradle project (File -> Import -> Existing Gradle Project)
3. Open the Cube.Java file and run the program
4. Review readme-users.txt for information on using the software
